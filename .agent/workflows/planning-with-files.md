# Planning with Files

This workflow manually invokes the `planning-with-files` skill to start a complex task using persistent file-based memory.

1. **Activate Skill**
   Start the `planning-with-files` skill.

   Task: "I want to start a complex project using the planning-with-files skill. Please initialize the necessary files (task_plan.md, findings.md, progress.md) and guide me through the planning process."

## Output

- Produce a concrete deliverable in markdown aligned with the workflow/skill goal.
- Include key decisions, actions taken, and final status for Antigravity IDE visibility.
